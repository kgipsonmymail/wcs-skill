# WCS 参考文档索引

本文件为 WCS 参考文档的指针索引。

## 文件

- `core_docs_template.md`：基线文档集合、必备章节与粒度说明
- `workflow_checklists.md`：从任务接受到交接的操作清单
- `coding_standard_template.md`：新项目可套用的可移植代码规范模板

## 加载指引

按任务类型有选择地加载：

- 新项目搭建：加载上述三个文件
- 功能开发：加载 `workflow_checklists.md` 与项目代码规范
- 缺陷修复：加载 `workflow_checklists.md`，并更新 `error_book` 相关段落
- 流程/治理调整：加载 `core_docs_template.md` 与 `coding_standard_template.md`
